# best-version-builder v1.3.3

A skill for comparing candidate versions, scoring them with evidence-grounded rubrics, improving the best candidate, validating the result, and handing off auditable artifacts without mixing internal working notes into release contents.

## Contents

- `SKILL.md` — skill definition
- `VALIDATION.md` — validation summary for this package

## Release-content hygiene

Delivered packages should describe behavior, usage, constraints, validation, and maintenance contracts. Comparison history, evaluator names, internal process labels, and rejected-option narratives belong in an audit packet unless explicitly requested.

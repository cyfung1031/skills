# RealityBench CaseForge v1.0.0

RealityBench CaseForge generates source-grounded, format-fit, first-shot-ready benchmark test cases that approximate non-synthesized real-world testing and produce globally differentiated, objectively scoreable evaluation suites.

v1.0.0 adds dedicated LLM benchmark validity controls for stochastic outputs, prompt sensitivity, context-window behavior, evidence grounding, tool use, safety/refusal boundaries, multilingual and locale behavior, cost pressure, and judge/scorer reliability.

## Core validity formula

```text
valid case = realistic + source-grounded + format-valid + globally differentiable + first-shot-ready + objectively scoreable + decision-relevant + LLM-characteristic-aware when applicable
```
